# Tools Required
- Homebrew
- Xcode

# Install Homebrew
```sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

# Install GLFW, GLEW and GLM
- Path for M1: /opt/homebrew/include
- Path for Intel: /usr/local/include

```sh
# install glfw
brew install glfw3

# install glew
brew install glew

# install glm
brew install glm
```
